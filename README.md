# Dynamic Stylz Salon — v2

Small, fast, static one-page site for Dynamic Stylz Salon (Elma, WA). No framework, no build step — just HTML, CSS, and a sprinkle of JS.

- `index.html` — page markup + JSON-LD
- `styles.css` — design tokens + sections
- `app.js` — mobile nav, year, scroll-reveal
- `media/` — photography + branding assets

Deployed on Vercel at `v2.dynamicstylz.com`.
